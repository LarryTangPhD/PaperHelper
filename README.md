# 📚 新传论文智能辅导系统

一个专为新闻传播学研究生设计的智能论文写作辅导平台，基于AI大模型技术，提供从选题到成稿的全流程学术支持。

## 🎯 系统特色

### ✨ 核心功能
- **智能选题指导**：选题生成、可行性分析、创新性评估
- **论文批注修改**：智能批注、修改建议、逻辑检查
- **格式规范修正**：多种格式标准、自动格式转换
- **学习助手**：文献管理、写作指导、进度追踪

### 🔧 技术亮点
- 基于通义千问大语言模型（默认）
- 支持多种AI模型切换
- 新闻传播学专业定制
- 智能提示词工程
- 实时反馈机制

## 📋 功能模块

### 1. 🎯 选题指导模块
- **选题生成器**：根据研究主题生成多个高质量选题
- **选题诊断器**：评估选题可行性、创新性、可操作性
- **可行性分析**：理论可行性、方法可行性、数据可获得性
- **研究趋势**：学科热点追踪、前沿研究方向

### 2. ✏️ 论文批注模块
- **智能批注**：对论文内容进行AI专业批注
- **修改建议**：提供具体的修改意见和改进方向
- **逻辑检查**：检查论文逻辑结构和论证过程
- **学术规范**：确保符合学术写作规范

### 3. 📐 格式修正模块
- **格式检查**：检查论文格式规范性
- **引用格式**：自动修正引用格式（APA、MLA、GB/T 7714等）
- **参考文献**：生成标准参考文献列表
- **排版优化**：优化论文排版和格式

### 4. 🎓 学习助手模块
- **文献管理**：文献检索、整理、引用管理
- **写作助手**：学术写作技巧指导、模板生成
- **进度追踪**：论文写作进度管理、时间规划
- **学术资源**：研究方法、理论资源、期刊推荐

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Streamlit 1.44.0+
- 其他依赖见 `requirements.txt`

### 安装步骤

1. **克隆项目**
```bash
git clone [项目地址]
cd PaperHelper
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **设置环境变量**
```bash
# 推荐使用通义千问（默认）
export DASHSCOPE_API_KEY="your_dashscope_api_key"

# 或者使用DeepSeek（备用）
export DEEPSEEK_API_KEY="your_deepseek_api_key"
```

4. **运行应用**
```bash
streamlit run PaperHelper.py
```

## 📖 使用指南

### 选题指导
1. 选择"选题指导"模块
2. 输入研究主题描述
3. 设置论文篇幅和创意程度
4. 选择研究类型
5. 点击"生成选题建议"
6. 查看选题诊断和可行性分析

### 论文批注
1. 选择"论文批注"模块
2. 粘贴论文内容
3. 选择批注类型
4. 点击"开始批注"
5. 查看详细批注结果

### 格式修正
1. 选择"格式修正"模块
2. 输入论文内容
3. 选择目标格式
4. 点击"开始格式修正"
5. 查看修正后的内容

### 学习助手
1. 选择"学习助手"模块
2. 根据需要选择相应功能
3. 获取写作指导、模板或资源推荐

### 前端技术
- **Streamlit**：Web应用框架

- **格式识别算法**：自动格式检测和转换
